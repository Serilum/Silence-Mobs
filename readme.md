<h2>Mod Description</h2>

<p style="text-align:center"><a href="https://curseforge.com/members/serilum/projects" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/curseforge.svg" width="200"></a> <a href="https://modrinth.com/user/Serilum" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/modrinth.svg" width="200"></a> <a href="https://patreon.com/serilum" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/patreon.svg" width="200"></a> <a href="https://youtube.com/@serilum" rel="nofollow"><img src="https://raw.githubusercontent.com/Serilum/.data-workflow/main/badges/svg/youtube.svg" width="200"></a></p>
<p><strong><span style="font-size:24px">Requires the library mod&nbsp;<a style="font-size:24px" href="https://curseforge.com/minecraft/mc-mods/collective" rel="nofollow">Collective</a>.<br></span></strong></p>
<p><span style="font-size:18px">Silence Mobs is a minimalistic mod that adds the functionality to silence a mob with <strong><span style="color:#fc0;font-size:18px">The Silence Stick</span></strong>, which can be collected by transforming a stick via the <em>/silencestick</em> (or <em>/st</em>) command.</span><br><br><br><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/Serilum/.information/wiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span><br></strong><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/silence-mobs/a.png" alt="Configurable" width="853" height="180"></picture><br><br><strong>How does it work?</strong><br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/silence-mobs/b.png" alt="The Silence Stick" width="854" height="480"></picture><br>While holding a stick, enter the&nbsp;<em>/silencestick</em> (or&nbsp;<em>/st</em>) command to receive <strong>The Silence Stick</strong>.<br><br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/silence-mobs/c.png" alt="Item description" width="401" height="133"></picture><br>What a beauty.<br><br><picture><img src="https://github.com/Serilum/.cdn/raw/main/projects/silence-mobs/d.png" alt="Sheep" width="854" height="480"></picture><br>Now hit any mob with it, and it'll be silenced. The "Silenced Entity" name feature is togglable in the config.<br><br><br><strong>Also works with named entities:<br></strong></p>
